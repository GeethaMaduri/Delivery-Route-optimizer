# Delivery Route Optimizer

## Overview

Delivery Route Optimizer is a Design and Analysis of Algorithms (DAA) project that compares Greedy, Dijkstra, and A* algorithms for route optimization.

The project uses a synthetic road network dataset and visualizes optimized delivery routes using Streamlit and Folium.

## Features

* Synthetic dataset generation
* Graph modeling
* Greedy Algorithm
* Dijkstra Algorithm
* A* Algorithm
* Algorithm comparison and validation
* Multi-stop delivery optimization
* Interactive Streamlit dashboard
* Folium map visualization

## Project Workflow

1. Generate synthetic delivery network
2. Build graph representation
3. Implement Greedy routing
4. Implement Dijkstra shortest path
5. Implement A* shortest path
6. Compare algorithm performance
7. Multi-stop route optimization

## Technologies Used

* Python
* Pandas
* Streamlit
* Folium
* Jupyter Notebook

## Run Application

```bash
streamlit run app.py
```
